Project: Local Network-Based SQLite Database with React and Express.js

Overview

This project provides a local network-based SQLite database system accessible through an Express.js API. It allows multiple devices connected to the same network to store and retrieve student attendance data without requiring an internet connection.

Features

SQLite Database: Lightweight and file-based database system.

Local Network Access: Runs on a LAN, making it accessible from multiple devices.

Express.js Backend: API built with Express.js to interact with SQLite.

Data Submission: Allows students to submit Name, ID, Code, and Date.

Data Retrieval: Fetch stored student data via API.

CORS Enabled: Ensures smooth communication between frontend and backend.

Firewall and Permission Setup: Configurations to allow network access.

Auto Table Creation: Automatically creates a database table if not already present.
